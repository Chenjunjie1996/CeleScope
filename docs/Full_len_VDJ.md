# Introduction
This pipeline is currently mainly for TCR/BCR full-length assembly. The whole process includes sample, barcode, convert, assemble, annotation, match, summarize, mapping. The sample and barcode steps are the same as the previous process. For related information, please refer to the description of CeleScope.
