Estimated Number of Cells: Cells with at least ine productive TRA or TRB chain.

Reads Mapped to Any V(D)J Gene: Reads mapped to any TCR or BCR genes.

Reads Mapped to TRA: Reads mapped to TRA chain. For BCR, this should be one of [IGH, IGL, IGK].

Reads Mapped to TRB: Reads mapped to TRB chain. For BCR, this should be one of [IGH, IGL, IGK].

Fraction Reads in Cells: Number of reads with cell-associated barcodes divided by the number of reads with valid barcodes.

Median TRA UMIs per Cell: Median number of UMIs assigned to a TRA contig per cell. For B cells, only the max of [IGK, IGL] are counted.

Median TRB UMIs per Cell: Median number of UMIs assigned to a TRB contig per cell. For B cells, only the max of [IGK, IGL] are counted.
